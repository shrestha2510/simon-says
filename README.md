# Simon Says Game

Simon Says is a classic memory-based game where players must repeat an increasingly complex sequence of colors or sounds. The game tests the playerâ€™s memory, concentration, and reaction skills in a fun and interactive way.

---

## Features

- Interactive and engaging gameplay
- Randomly generated color sequences
- Increasing difficulty with each level
- Visual and/or sound-based cues
- Instant feedback on correct or incorrect inputs
- Simple and user-friendly interface

---

## How the Game Works

1. The game starts with a single color in the sequence.
2. The player must repeat the sequence by clicking the correct colors in order.
3. Each successful round adds a new color to the sequence.
4. The game continues until the player makes a mistake.
5. The final score or level achieved is displayed.
